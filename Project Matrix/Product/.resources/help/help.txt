help           - Mostra esse menu:
info *função*  - Mostra detalhes de uma função especifica
minfo          - Mostra detalhes da matriz
clear          - Limpa a tela
show/hide      - Deixa a mostra a matriz em X ou a esconde
quit           - Fecha programa
intro          - Re-mostra a intro

Alocação de matrizes  | Operações               | Editar X
                      |                         | 
E  - Enter            | sum - Somar       (Y+X) | size   - Dimensões da matriz
D  - Drop             | sub - Subtrair    (Y-X) | choose - Inserir valores em sequencia
S  - Swap             | tim - multiplicar (Y*X) | point  - Inserir valores em coordenadas
R  - Roll             |                         | rand   - Inserir valores aleatórios
AC - Reseta memória   |                         | tr     - Transpor matriz
                      |                         | inv    - Inverter matriz
                      |                         | clr    - Reseta matriz
