int  l,
char  txt[100] 
)

Imprime texto inserido dentro de uma caixa.

Esta funcao tem papel estetico de introduzir texto posto sobre ela dentro de uma caixa de largura e altura definida.

Parameters
hDefine a altura da caixa.
lDefine a largura da caixa.
txttexto que sera inserido dentro de uma caixa.
A finalização de uma linha deve ser indicada por "\n".
Returns
imprime na tela o texto inserido dentro de uma caixa.
Warning
Dimensoes minimas no input - l = 6 + numero de caracteres da maior linha do texto.
h < 4 + numero de linhas do texto, imprime o tamanho minimo da caixa para o texto inserido.

showhide, sh_matriz, sh_help,, info, interface.

void minfo ( void  )

Mostra detalhes sobre a matriz.

Calcula e mostra estatisticas referentes a matriz presente em X.
Tais estatisticas são:
- Dimensões da matriz;
- Formato da matriz;
- Tipo de matriz;
- Determinante*;
- Nível de inicialização da matriz.

Warning
*Não calcula determinante de matrizes quadradas de ordem > 3 !.
