Altera um valor em uma coordenada especifica em X.

Insere diretamente o valor fornecido pelo usuário as coordenadas selecionadas por ele.
Caso selecione uma coluna o valor é inserido sobre todas as linhas da mesma,
o contrário trocando linha por coluna tambem é verdadeiro,
e se nao inserir nenhuma coordenada a matriz é preechida
completamente com o valor inserido.
