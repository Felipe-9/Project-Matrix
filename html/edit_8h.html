◆ size()

void size ( void  )

Altera as dimenções de uma matriz.

Funções de edição da matriz localizada em X.

Se não existir matriz no local selecionado cria uma matriz apenas com dimenções definidas, necessitando atribuir a ela valores.
Se existir, altera as dimenções desta e caso as novas dimenções forem maiores que a anterior, sera necessario atribuir a ela valores a tais novos espaços.

Inicializa parcialmente a matriz no local selecionado.
